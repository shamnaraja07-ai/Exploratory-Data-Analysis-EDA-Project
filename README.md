**📊 Student Performance Exploratory Data Analysis (EDA)**



📌 Project Overview



This project performs Exploratory Data Analysis (EDA) on student academic performance data to identify trends, relationships, and factors influencing student outcomes.



The analysis uses statistical summaries and visualizations to better understand the dataset and derive meaningful insights.







🎯 Objectives



\- Analyze student academic performance data

\- Identify patterns and correlations

\- Visualize important trends

\- Discover key influencing factors

\- Improve analytical and data exploration skills

📊 Dataset Description



The dataset contains information related to student academic activities and performance.



| Column Name | Description |

|-------------|-------------|

| StudentID   | Unique student identifier |

| StudyHours  | Average study hours per day |

| Attendance  | Attendance percentage |

| AssignmentsCompleted | Number of assignments completed |

| InternalMarks | Internal assessment marks |

| PreviousSemesterScore | Previous semester performance |

| ProjectScore | Academic project evaluation score |

| Result | Final result (0 = Fail, 1 = Pass) |

📈 Visualizations Generated



1\. Correlation Heatmap

2\. Attendance Distribution

Displays attendance frequency among students.



3\. Study Hours vs Internal Marks

Analyzes the impact of study hours on academic performance.





🔍 Key Insights



\- Higher attendance generally improves academic performance.

\- Students who complete more assignments tend to achieve better results.

\- Internal marks are strongly related to study hours.

\- Previous semester performance influences current results significantly.
